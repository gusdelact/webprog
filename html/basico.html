Primer parrafo

Un documento HTML debe ser escrito bajo las siguientes reglas:

  • Siempre debe empezar con doctype
  • La etiqueta raíz es html
  • Se debe anidar head y body a la etiqueta html
  • Elemento head debe tener m&icute;nimo la etiqueta title y meta
  • Elemento body anida cualquier etiqueta (excepto) algunas de head (como title o meta)
  • Nunca poner un texto libre a nivel de body
  • Siempre etiquetar un texto con alguna etiqueta, por ejemplo p
  • No olvidar cerrar etiquetas
  • Un elemento HTML vacio es aquel que abre y cierra la etiqueta, sin contener texto. Ejemplo es br o img
  • Cuando la etiqueta lleve un atributo, su valor siempre va con doble comilla y no olvidar cerrar dicha doble comilla.
  • Atributos tipicos para TODAS las etiquetas son el atributo id, class, style
  • Usar siempre minúsculas para nombrar etiquetas HTML
